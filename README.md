
# 🚀 HACKING PROTOCOL

### A Neon Cyber-Themed Grid Puzzle Game

**HACKING PROTOCOL** is a visually engaging, grid-based puzzle game built using **Python** and **Pygame**.
Navigate through a cyber network, unlock gates, collect power nodes, and avoid intelligent virus enemies.

---

## 🎮 Features

* ⚡ Neon cyberpunk visual theme
* 🧠 Smart enemy AI with pathfinding
* 🔐 Gate unlocking system using linked nodes
* 🔋 Power-up mechanics (temporary abilities)
* 🧩 Multiple levels with increasing difficulty
* 🎯 Real-time player movement & interactions

---

## 🕹️ Gameplay Overview

* You control a player navigating a network grid.
* Collect **Power Nodes** to activate abilities.
* Use power to unlock **Gates** via **Bulb Nodes**.
* Avoid or destroy **Viruses**.
* Reach the **Exit** to complete each level.

---

## 🎹 Controls

| Key   | Action        |
| ----- | ------------- |
| W / ↑ | Move Up       |
| S / ↓ | Move Down     |
| A / ← | Move Left     |
| D / → | Move Right    |
| R     | Restart Level |
| N     | Next Level    |

---

## 🛠️ Tech Stack

* **Language:** Python 3.x
* **Library:** Pygame
* **Concepts Used:**

  * Object-Oriented Programming (OOP)
  * Breadth-First Search (BFS) for AI pathfinding
  * Game loop & event handling
